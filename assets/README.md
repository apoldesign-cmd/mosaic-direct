# Assets

## hero-banner.jpg

Coloque a imagem do banner (campo de soja com sunset + logo Mosaic) neste arquivo:
`/assets/hero-banner.jpg`

Referenciada via CSS nas 8 telas (`background-image: url("./assets/hero-banner.jpg")`).
Se o arquivo não existir, o gradient antigo aparece como fallback.

Tamanho recomendado: 1920x300px ou similar (formato horizontal, alta resolução).
