Como navegar

  • Cada card abre o protótipo correspondente em uma nova janela do mesmo link.
  • Em qualquer tela, use o botão flutuante 🧪 Protótipos no canto inferior esquerdo para alternar.
  • O botão atual fica destacado em verde no menu.
  • Pressione Esc ou clique fora para fechar o menu.

Sobre os mockups

  • Todos os dados são fictícios e servem só para ilustrar o fluxo.
  • Os contratos disponíveis (Q-20303, Q-20385, Q-20392, Q-38542, Q-85721, Q-39742) repetem entre as telas para consistência.
  • Tipografia Manrope, paleta oficial Mosaic.
  • Layout calibrado para 1320px de largura útil.
