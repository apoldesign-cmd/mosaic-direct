Como medimos. Carregamento = melhor de 3 medições do evento load de cada tela em ambiente local (localhost), com o JavaScript executando. Contagens (nós, campos, botões, colunas, elementos interativos) extraídas do DOM renderizado e dos arquivos. Cliques = caminho mínimo (“caminho feliz”) da tarefa principal de cada tela. São números do protótipo, não de produção — servem para comparar as telas entre si e dimensionar a solução.
